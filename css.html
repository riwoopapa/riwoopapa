css에 사용되는 태그 정리

주요내용 메모

css를 적용시키기 위해서 여는style태그와 닫는/style태그를 사용
style "font-size: 20px;" 와 같은 방식으로 폰트사이즈 변경

모든 p태그 단락에 스타일 적용이 가능한데
head태그 요소 안에 반드시 존재하여야하며

style
p{font-family: arial}
/style
와 같은 방식으로 모든 p태그 요소의 스타일 변경 가능

.css 확장자를 사용하여 따로 저장이 가능한데 이는, style태그를 제외한 안의 내용만을 작성하여 .css확장자로 저장하면 됨

.css파일과 .html파일을 연계하여야 스타일 적용이 되는데 이는 link태그를 사용하며, 이는 클로징태그는 없다

link태그는 href , type, rel 요소를 필요로함 (반드시 head태그 안에 위치)
href는 주소(https://..... , ./index.css , 등)
type은 파일의 속성 (text/css)
rel은 연계속성 (stylesheet)

모든 태그에 class속성을 추가함으로써 스타일을 지정할수 있다
h1 class="green" 이런 방식으로.

.css파일에서 여기에 스타일을 적용시키려면 .green{color=red;} 이런식으로 입력

h1 id="green" 이런 방식으로

.css파일에서 여기에 스타일을 적용시키려면 #green{font-family: arial;} 이런식으로 입력

class요소와 id요소의 다른점은 특이성이다. 일반태그{class{id 순서로 모두다 색상을 정의했을때, id에 해당하는 색상이 출력된다.

체인태그도 가능하다
예를들어 h2.green{font-famliy: cursive;} 이런식으로 h2와 green class모두 한꺼번에 사용하여 변경이 가능하다.

font-family: georgia, helvetica, "courier new", garamond, times new roman(기본)

text-align: left, right, center

.main-banner { background-image: url("https://www.example.com/image.jpg"); }
와 같은 방식으로 백그라운드 이미지 삽입가능

font-weight: bold, thin, nomal

[ box model ]
height
width
padding
border
margin
overflow

border
: width(thin, medium, thick)
: style(none, dotted, solid)
: color
ex) p { border: 3px solid coral; }

칼라키 목록은 다음의 링크에 잘나와있음
https://developer.mozilla.org/en-US/docs/Web/CSS/color_value

테두리 스타일 목록은 다음의 링크에 잘나와있음
https://developer.mozilla.org/en-US/docs/Web/CSS/border-style#Values

임의로 만들어진 테두리는 {border: medium none color;}

border-radius: 0px; 혹은 0%; 로 모서리를 둥글게 만들어줄수있음

padding: 0px; 로 테두리와 컨텐츠 사이의 간격 조절
padding-top
padding-right
padding-bottom
padding-left
로 정확한 속성 조정가능

padding: 6px 11px 4px 9px; 로 간단하게 세부조정가능
top (6 pixels), right (11 pixels), bottom (4 pixels), left (9 pixels)

padding: 5px 10px; 로 간단하게 세부조정가능
top & bottom (5px), right & left (10px)

margin: 0px;로 박스이외의 여분을 설정가능
margin-top
margin-right
margin-bottom
margin-left
로 세부 조정가능

margin: 5px 10px; 로 간단하게 세부조정가능
top & bottom (5px), right & left (10px)

margin: 0 auto; 를 사용함으로써 가운데에 위치하게 할 수 있다


min-width: 0px;
max-width: 0px; 너비를 최소~최대까지 설정
min-height: 0px;
max-height:0px; 높이를 최소~최대까지 설정

overflow 위값을 사용해서 컨텐츠가 넘치는 오류가 발생했을때 제어하는 태그
hidden, scroll, visible(기본)

패딩 및 마진 초기화는 padding: 0; margin 0;으로 설정

visibility로 컨텐츠를 숨길수 있다
hidden, visible
(display: none;과의 차이는 visibility는 단순히 숨겨줄뿐 내용은 존재한다는 차이)
